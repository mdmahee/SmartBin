# SmartBin
IOT Based Smart Bin Management system.

Web Application: PHP laravel.
Database: MySQL.
Communicaion: GSM/GPRS.
MCU: ATmega328p.
Sensor: Sharp IR Proximity Sensor,
        Mercury tilt sensor,
        Multi-lavel Humidity Sensor.
Status: RGB LED Indicator.
Power Supply: 3.7v,2000mA.
GSM Module: sim800.
